# 关于本站

> 这个网站是我用 Nuxt 3 搭建的，现已开源在 <a href="https://github.com/2696791698/www.houning.top" rel="noopener noreferrer" target="_blank">Github</a> 上。

这是我的个人网站，用来记录折腾过程、学习笔记和阶段性作品。

本站以「可读性优先」为核心，追求页面简洁、加载轻量、交互自然。

本站使用的技术栈：

- Nuxt 3 / Vue 3 + @nuxt/ui + @nuxt/content
- Tailwind CSS 4

站点还在持续优化中。
