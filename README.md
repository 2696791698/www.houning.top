我的个人网站, 基于 Vue 3 生态系统构建.

## 技术栈

- **框架**: Nuxt 3
- **样式**: Tailwind CSS 4

## 本地运行

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```
